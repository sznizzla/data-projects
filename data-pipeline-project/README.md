End-to-End Data Pipeline (API → Database → Dashboard)

Overview

This project demonstrates a complete data engineering pipeline that collects data from a public API, processes it using Python, stores it in a SQL database, and visualizes insights through a dashboard.

The goal of this project is to simulate a real-world data workflow used in modern analytics platforms.

Architecture

API → Python ETL Pipeline → PostgreSQL Database → Dashboard Visualization

Technologies Used
	•	Python
	•	Pandas
	•	PostgreSQL
	•	SQL
	•	Airflow (optional)
	•	Power BI / Tableau

Features
	•	Automated data ingestion from an external API
	•	Data cleaning and transformation
	•	Structured storage in a relational database
	•	Analytical dashboard for insights
