Cloud Data Pipeline on Google Cloud

Overview

This project demonstrates a cloud-based data engineering pipeline built on Google Cloud. The system ingests data, stores it in cloud storage, processes it, and loads it into a data warehouse for analytics.

Architecture

Data Source → Cloud Storage → Data Processing → BigQuery Data Warehouse → Analytics

Technologies Used
	•	Google Cloud Platform
	•	BigQuery
	•	Python
	•	SQL
	•	Cloud Storage

Features

	•	Data transformation pipeline
	•	Analytical queries on BigQuery
	•	Cloud-native architecture
